`@memoir/tree` ships a small framework-free stylesheet for the intended Memoir skin: warm canvas, light cards, sharp corners, black outlines, and orange selected state.

## Import Styles

Import the stylesheet once for the default selectors and Memoir theme variables:

```tsx
import "@memoir/tree/styles.css";
```

You can override the variables or replace the card markup.

## Memoir Look

Use this as the starting point for flat product UIs:

```css
.memoir-tree {
  --tree-surface-bg: #fafafa;
  --tree-canvas-bg: #F4EFDC;
  --tree-card-bg: #fafafa;
  --tree-card-fg: #030201;
  --tree-card-border: #030201;
  --tree-card-selected-bg: #EC5A44;
  --tree-card-shadow: 4px 4px 0 #030201;
  --tree-edge: #030201;
  --tree-edge-width: 1;
  --tree-card-radius: 0;
  --tree-profile-radius: 0;
}
```

Then pass the surface class and a card component:

```tsx
function ProfileCard({
  collapsed: _collapsed,
  focused: _focused,
  onAddRelationship: _onAddRelationship,
  person,
  personId: _personId,
  readOnly: _readOnly,
  relation,
  selected: _selected,
  ...props
}: FamilyCardProps<Profile>) {
  return (
    <article {...props} className="memoir-profile-card">
      <strong>{person.name}</strong>
      <small>{relation.label}</small>
    </article>
  );
}

<FamilyTree
  className="memoir-tree"
  people={people}
  relationships={relationships}
  subject={personId}
  card={ProfileCard}
/>
```

## Theme Prop

For small changes, use `theme` instead of writing CSS variables manually:

```tsx
<FamilyTree
  people={people}
  relationships={relationships}
  subject="alex"
  theme={{
    surfaceBackground: "#fffdf4",
    canvasBackground: "#f4efdc",
    cardBackground: "#fffdf4",
    cardBorder: "#030201",
    cardRadius: 0,
    cardShadow: "6px 6px 0 #030201",
    edge: "#3b342e",
    edgeWidth: 2,
    cardSelectedBackground: "#ec5a44",
  }}
/>
```

Use `theme="memoir"` for the package default or `theme="system"` for neutral system colors.

## Selectors

The family renderer emits these selectors:

- `data-tree-surface`: outer scroll/pan surface.
- `data-tree-canvas`: positioned canvas used by the viewport.
- `data-family-canvas`: measured family canvas that contains edges and cards.
- `data-family-tree`: family tree canvas marker.
- `data-family-card-positioner`: absolute-positioned wrapper for a card.
- `data-family-measure`: measurement wrapper around a rendered card.
- `data-family-card`: card props forwarded to your custom card.
- `data-person-id`: current person ID.
- `data-relation`: computed relation label like `self`, `parent`, or `half-sibling`.
- `data-generation`: generation relative to the subject.
- `data-side`: `ancestor`, `descendant`, `sibling`, `partner`, or `self`.
- `data-selected`: present on the selected card.
- `data-focused`: present on the focused card.
- `data-family-edge`: SVG edge path.
- `data-source-id`: source person ID for an edge.
- `data-target-id`: target person ID for an edge.
- `data-edge-kind`: relationship kind like `biological`, `adoptive`, `spouse`, or `guardian`.
- `data-edge-status`: relationship status like `current`, `former`, or `divorced`.

```css
[data-family-card][data-relation="half-sibling"] {
  background: #ffea00;
}

[data-family-edge][data-edge-kind="adoptive"],
[data-family-edge][data-edge-status="former"] {
  stroke-dasharray: 4 4;
}
```

## Card Root

Your card receives normal HTML props plus tree metadata. Always spread the props onto the root element so selection, data attributes, click handlers, and accessibility metadata reach the DOM:

```tsx
function ProfileCard({
  collapsed,
  person,
  personId,
  readOnly,
  relation,
  selected,
  onAddRelationship,
  ...props
}: FamilyCardProps<Profile>) {
  return (
    <article {...props} className="memoir-profile-card">
      <strong>{person.name}</strong>
      <small>{selected ? "selected" : relation.label}</small>
      {!readOnly ? (
        <button type="button" onClick={() => onAddRelationship?.(person, personId)}>
          Add relative
        </button>
      ) : null}
      {collapsed ? <span>Collapsed</span> : null}
    </article>
  );
}
```

For extra typed props, use `cardProps` on `FamilyTree`. See [FamilyTree](./family-tree) for the full card contract.

```tsx
<FamilyTree
  people={people}
  relationships={relationships}
  subject={personId}
  card={ProfileCard}
/>
```
